## Getting Started

### Yang perlu diinstal
- nodejs
- git

sambungin git dan github, terus run command ini di terminal (pastikan path terminal ada di folder yang mau disimpen file code nya)

```bash
git clone https://github.com/arya-satwika/tutorly.git

npm install package.json package-lock.json

npm run dev
```

buka [http://localhost:3000](http://localhost:3000) di browser buat liat hasil editan code

setelah code udh diedit dan mau di save, commit semua changes di vscode lalu push (sync changes)
